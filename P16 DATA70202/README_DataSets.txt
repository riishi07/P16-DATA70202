README: Dataset Descriptions
=============================

This file provides an overview of the dataset structure and contents for the project 
"Network Analysis of Algal Blooms in UK Lakes".

--------------------------------------------------------------------------------
1. Data Sources
--------------------------------------------------------------------------------

Due to file size limitations, the original datasets are **not included** in this repository.

However, they are **publicly available** and can be accessed from the UK Centre for Ecology & Hydrology:

üèõÔ∏è **Source**: UK Centre for Ecology & Hydrology (UKCEH)

These include long-format CSVs of monthly or high-frequency environmental monitoring data from:

- Windermere South Basin
- Windermere North Basin
- Esthwaite Water
- Blelham Tarn

Data types:
- Nutrients (Nitrate, Phosphate, etc.)
- Phytoplankton groups (Diatoms, Cyanobacteria, Green Algae)
- Meteorological variables (Solar radiation, Temperature, Rainfall, Wind Speed)
- Hydrological indicators (Water Temperature, Oxygen, Secchi Depth, Stratification)

Note: These files may have missing values and require preprocessing before analysis.
